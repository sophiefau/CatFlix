CatFlix

This project is a small web application, showcasing movies featuring cats.
The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Request body data format Response body data format
Return a list of all movies. /movies GET None A JSON object holding data about all movies.
Return data about a single movie by title. /movies/:title GET None A JSON object holding data about a single movie including title, description, genre, director, featured cats, image URL. Example:

Title: "The Aristocats"
Director: "Wolfgang Reitherman"
Genre: "Animation, Adventure, Comedy"
Year: "1970"
Cat: "Duchess (mother) and her kittens: Marie, Berlioz, and Toulouse."
Synopsis: "A wealthy old woman plans to leave her fortune to her cats, but her butler tries to get rid of them. The cats must find their way back home with the help of a charming alley cat." Image: "img/the-aristocats.jpg"
Return data about a genre, such as genre description (e.g., “Thriller”). /genres/:genre GET None A JSON object holding data about then genres of the movie. Example:
Genre: "Thriller"
Description: "Thrillers are very thrilling."
Return data about the featured cat (name, color/breed, bio) by name. /movies/cat/:catName GET None A JSON object holding data about the featured cat-s including name, color/breed and bio. Example:
Name:"Duchess (mother) and her kittens: Marie, Berlioz, and Toulouse."
Color/Breed: "Duchess (white Persian), Marie (white Persian), Berlioz (gray kitten), Toulouse (orange kitten)."
Bio: "Duchess is a refined, elegant mother cat who lives in Paris with her three kittens. Marie is sweet and ladylike, Berlioz is musically inclined, and Toulouse is artistic and adventurous."
